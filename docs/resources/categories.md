# Pokémon Categories Resource

## Get Pokémon Categories
<span class="resource"><span class="get">GET</span> /categories</span>

Returns an array of Pokémon Categories discovered in the Pokémon World.

### Example Response
```json
[
  "starter",
  "legendary",
  "mythical",
  "ultraBeast",
  "mega"
]
```
